HTML5:

HTML5 ist die fünfte Version der Hypertext Markup Language (HTML), die die Struktur und das Layout von Webinhalten definiert. Es verbessert die Unterstützung für Multimedia, grafische Inhalte und interaktive Elemente ohne zusätzliche Plugins. Neue Elemente wie <canvas>, <video> und <audio> sowie erweiterte Formularelemente und verbesserte Semantik sind enthalten.

CSS3:

CSS3 ist die neueste Version der Stylesheet-Sprache, die das Erscheinungsbild von Webinhalten gestaltet. Es bringt neue Funktionen wie erweiterte Selektoren, Media Queries für responsives Design und visuelle Effekte wie Schatten, Animationen und Übergänge. Diese Verbesserungen ermöglichen ansprechendere und dynamischere Websites.

JavaScript:

JavaScript ist eine vielseitige Programmiersprache, die zur Erstellung interaktiver Webinhalte verwendet wird. Zusammen mit HTML und CSS animiert sie Webseiten, verarbeitet Benutzereingaben und aktualisiert Inhalte ohne Seitenneuladung. JavaScript unterstützt objektorientierte, imperative und funktionale Programmierung und ist eine Kerntechnologie des modernen Webs.

MariaDB:

MariaDB ist ein Open-Source-Datenbankverwaltungssystem, das als Fork von MySQL entwickelt wurde. Es bietet hohe Leistung, Skalierbarkeit und Sicherheit und ist vollständig kompatibel mit MySQL. MariaDB wird oft für webbasierte Anwendungen und Services verwendet, die zuverlässige Datenbanksysteme benötigen.

C#:

C# ist eine objektorientierte Programmiersprache, die von Microsoft im Rahmen der .NET-Initiative entwickelt wurde. Sie kombiniert die Rechenleistung von C++ mit der Einfachheit von Visual Basic und bietet eine starke Typensicherheit und umfassende Standardbibliothek. C# wird häufig für Windows-Anwendungen, Webservices und Spieleentwicklung verwendet.
